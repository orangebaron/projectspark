Projects

India Pilot Program

  • We provided students in India with computers, keyboards, mice, and screens.
  • We provided them with internet access.
  • We instructed teachers on how to teach with the resources given.
  • We provided them with the knowledge needed to teach students Scratch, a simple programming language.
  • The project was a success, and we hope to do an improved version of this project in more schools very soon.

Library Classes

  • We instructed students in scratch in a 4-week-long programming class at our local library.
  • The project was successful, and we will do another library class next summer.
  • We are making our curriculum publically available here.
